Variable Names

  1. Use semantic variable names
  2. Adopt useful conventions
    1. postfix color nomes
    2. prefix specific sections
  3. Keep a centralized config

Semantic Names

// Bad
$red: red;
$yellow: yellow;

// Better
$brand-color: red;
$accent-color: yellow;

Adopt useful conventions

// Base colors
$base-color: #333;
$brand-color: red;
$brand-80-color: rgba($color-brand, 0.8);
$accent-color: yellow;

// Header
$header-height: 100px;
$header-background-color: $color-brand;

// Footer
$footer-height: 200px;
$footer-background-color: #aaa;

Keep a centralized config

@import base/config;
@import base/typography;
@import base/utilities;

@import modules/button;
